# island-cli
> Island blog system command line tool

## Install

```
$ npm install -g island-cli
```

## Usage

```
Initialize a blog
    $ island -i
    $ island init

Show current version
    $ island -v
    $ island version

Add a new article
    $ island -a <name>
    $ island add <name>

Delete an article
    $ island -x <name>
    $ island del <name>

Reset all data
    $ island -r
    $ island reset

Update a theme
    $ island -u <name>
    $ island update <name>

Development
    $ island -d
    $ island dev

Production
    $ island -b
    $ island build
```
